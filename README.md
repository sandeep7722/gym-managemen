# Gym Management App

A React-based gym management application with owner login functionality.

## Features

- Owner login using owner ID or phone number and password
- Home page after successful login

## Installation

1. Ensure you have Node.js version 16 or higher installed.
2. Clone the repository.
3. Run `npm install` to install dependencies.

## Running the Application

- Development: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`

## Project Structure

- `src/App.jsx`: Main app component with routing
- `src/components/Login.jsx`: Login page component
- `src/components/Home.jsx`: Home page component
- `src/data/ownerDetails.js`: Mock owner data

## Usage

1. Start the development server.
2. Navigate to the login page.
3. Enter owner ID (e.g., "1") or phone number (e.g., "1234567890") and password (e.g., "password123").
4. Upon successful login, you will be redirected to the home page.
